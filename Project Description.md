//## Online Learning System – Project Description

//The **Online Learning System** is a simple Java-based application developed using Object-Oriented Programming (OOP) concepts. This project allows students to view available courses, enroll in courses of their choice, and manage their enrolled courses through a menu-driven console interface. The system is designed using multiple classes such as Student, Course, and LearningSystem to ensure modularity and proper object-oriented structure. It demonstrates how Java OOP principles can be used to build a basic learning management system that is easy to use, scalable, and organized.

//### Key Features

//* View available courses
* Enroll in preferred courses
* Display enrolled courses
* Menu-driven console interface
* Simple and user-friendly design

//### OOP Concepts Used

* Classes and Objects
* Encapsulation
* Constructors
* Method calling between classes
* ArrayList for storing data
* Modular program structure
